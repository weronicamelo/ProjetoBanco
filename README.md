# Projeto Banco

Aplicação console para conta de banco que permite o saque, depósito e transferência.

## Funcionalidades

- Cria uma conta bancária
- Faz saque 
- Faz depósito 
- Faz transferência

## Tecnoligias utilizadas

- dotnet 6.0

## Como executar

Realize o clone do repositório e execute o comando: 
```
dotnet run
```
